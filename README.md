# Cuối Tuần Đi Đâu - Android APK

Repo này dùng để phân phối file APK ứng dụng Android cho [Cuối Tuần Đi Đâu](https://cuoituandidau.vn/).

## Tải về

Tải file APK mới nhất tại mục [Releases](../../releases).

## Hướng dẫn cài đặt

1. Tải file `.apk` từ mục Releases về điện thoại Android
2. Mở file `.apk` vừa tải
3. Nếu được hỏi, cho phép cài đặt từ nguồn không xác định (Unknown Sources) trong Cài đặt
4. Nhấn **Cài đặt** và chờ hoàn tất

## Yêu cầu

- Android 7.0 trở lên
